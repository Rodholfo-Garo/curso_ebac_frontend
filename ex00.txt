O HTML (Hiper Text Markup Language) é uma linguagem de marcação de texto, ela é a estrutura básica de um sistema WEB. Em 2014 foi lançado o HTML5, que trouxe mais vantagens em seu uso para WEB, como a possibilidade de desenhar componentes gráficos , como caixas, círculos, texto e imagens com o uso do elemento <canvas>. O HTML trabalha em conjunto como CSS (CASCATING STYLE SHEET), que aplica estilo ao documento HTML. Através do CSS, definimos a aparência da página, sua adaptaço a divérsos tipos de tela (site responsivo).
O Java Script é uma linguagem programação de alto nivél, responsavel por deixar as paginas dinamicas e interativas. Foi criado em 1995 por Brendan Eich. Esta linguagem pode ser usada tanto no Front-End quanto no Back-end.
